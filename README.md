## Como Usar

1. **Substitua a foto do casal:**
   - Coloque sua foto em `images/` 
   - Atualize o nome no HTML se necessário

2. **Abra o site:**
   - Clique duas vezes em `index.html`
   - O site abrirá no seu navegador
